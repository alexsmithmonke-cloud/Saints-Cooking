# Saint's Cooking

This is a dev friendly cooking addon.
To add or change any recipe, simply head to the sh_saintscooking.lua file and use the templates given, there.
You *NEED* to make your own things in sh_saintscooking.lua or download the many prop addons in there such as foodhouseholditems for the sake of models.

Press F4 to spawn in items related to the addon.
